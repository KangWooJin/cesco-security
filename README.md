### 스터디 방법
9주 완성으로 진행하며, 매주 한 명씩 돌아가면서 수업을 리딩한다.

- 발표자의 수업 리딩(1시간)

- 발표자의 과제 리딩 (1시간)

수업 진행은 자유 방식이다.

과제는 간단하게 시큐리티 구현 or 테스트 코드 검증으로 만든다.
